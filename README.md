# PaintFX
Just a small Paint program I wrote, when experimenting with JavaFX
(Created with IntelliJ IDEA Community Edition

# Features
Draw:
- Freehand
- Ellipse
- Rectangle
- Line

Colors:
- Fill color
- Line color
- With/without fill

Additional:
- Move shapes
- Move shapes to front/back
- Undo/Redo (applies ONLY to adding/removing shapes, NOT to moving)
